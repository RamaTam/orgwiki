1 查看系统信息

1.1 查看内核

uname -a

1.2 查看系统是32位还是64位

#查看long的位数，返回32或64
getconf LONG_BIT
#查看文件信息，包含32-bit就是32位，包含64-bit就是64位
file /sbin/init

1.3 查看Ubuntu版本

cat /etc/issue
或 cat /etc/lsb-release 
或 lsb_release -dsc

1.4 查看内核加载的模块

lsmod

1.5 查看CPU信息

cat /proc/cpuinfo

1.6 显示当前硬件信息

sudo lshw

1.7 显示当前内存大小

free -m |grep "Mem" | awk '{print $2}'

1.8 查看硬盘温度

sudo apt-get install hddtemp
sudo hddtemp /dev/sda
